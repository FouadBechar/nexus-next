# Roadmap

Nexus Next is currently in beta with the core chat loop working in production.

## Current Beta

- Multi-model chat through OpenRouter
- Streaming responses
- Clerk authentication
- Supabase persistence for signed-in chats and messages
- Local persistence fallback for signed-out users
- Search across loaded chat titles and messages
- Settings for model, temperature, and system prompt
- Markdown and JSON chat export
- Mobile-friendly chat layout
- Vercel deployment

## Next

- File and image attachments
- PDF export
- Theme controls
- Supabase-backed full-text search for larger histories
- Better model metadata, badges, and capability indicators

## Later

- RAG over uploaded files
- Agent workflows
- Shared/public chat links
- Landing page polish
- Usage tracking and cost visibility
